Features
The terrain tool automates placement of transition tiles. This example uses tiles from the Liberated Pixel Cup (CC-BY-SA 3.0 ∓ GPLv3).

Efficient tile editing tools like stamp, fill and terrain brushes.
Demonstrating support for custom objects alongside a tile map with background image. Graphics from GfxLib-Fuzed (CPL).

Rectangle, ellipse, polygon and image objects can be placed with pixel precision.
An isometric map with a custom object, showing object shapes are projected as well. The tiles are by Clint Bellanger (CC-BY 3.0).

Supports orthogonal, isometric and staggered maps.
